# DevOps Learning Journey

My hands-on DevOps learning repository - exercises, projects, and notes from my DevOps training program and self-directed study.

## Repository Structure

```
devops-learning/
├── kubernetes/          # Kubernetes manifests & exercises
│   ├── 01-basics/       # Pods, namespaces, labels
│   ├── 02-deployments/  # Deployments, ReplicaSets, rolling updates
│   └── 03-services/     # Services, Ingress, networking
├── docker/              # Dockerfiles & container exercises
├── terraform/           # Infrastructure as Code with Terraform
├── bash-scripts/        # Automation & utility scripts
├── LEARNING_LOG.md      # Chronological log of completed exercises
└── README.md
```
